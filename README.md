# 📌 API de Clientes

A **API de Clientes** é uma aplicação desenvolvida para gerenciamento de clientes, permitindo realizar operações de cadastro, consulta, atualização e exclusão de registros.

O projeto tem como objetivo demonstrar a integração entre **backend, frontend e banco de dados**, utilizando uma arquitetura simples e organizada para facilitar manutenção, escalabilidade e boas práticas de desenvolvimento.

---

# 🚀 Tecnologias Utilizadas

## 🔧 Backend
- Java
- Spring Boot
- Spring Data JPA
- Maven

## 🗄 Banco de Dados
- PostgreSQL

## 🛠 Ferramentas de Desenvolvimento
- IntelliJ IDEA

---

# ⚙ Funcionalidades

A API permite realizar as seguintes operações:

- ✅ Cadastro de clientes  
- 📋 Listagem de clientes  
- ✏ Atualização de dados do cliente  
- ❌ Exclusão de cliente  
- 🔗 Integração entre API REST e frontend Angular  
- 💾 Persistência de dados no PostgreSQL

# 💡 Objetivo do Projeto

Este projeto foi desenvolvido para fins de **aprendizado e prática de desenvolvimento backend com Java e Spring Boot**, demonstrando a criação de uma **API REST integrada com banco de dados PostgreSQL e preparada para comunicação com aplicações frontend**.
